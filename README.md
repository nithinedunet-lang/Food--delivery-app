# Food Delivery App

Multi-restaurant ordering system with cart and order status tracking.

## Technologies Used
- Figma (UI/UX design)
- HTML
- Tailwind CSS
- Bootstrap

## Features
- List multiple restaurants and their menus
- Add items from different restaurants to the cart
- Place order and track order status

## Usage
1. Clone the repo.
2. Open `index.html` in your browser.
3. Use Figma assets in the `assets/images` folder.

## Customization
- Update restaurant information in `index.html`
- Configure cart and order tracking logic in `assets/js/cart.js` and `assets/js/order.js`
